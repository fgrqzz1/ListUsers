# 2. Мини-API «Список пользователей»
## Задача
Создать серверное API и клиент для получения списка пользователей.

## Функционал
- Сервер возвращает массив пользователей (JSON)
- Клиент делает GET-запрос
- Данные выводятся в таблицу
- Обработка ситуации, если сервер недоступен

## Технические требования
- REST-endpoint: /users
- Использование fetch()
- Асинхронный код (async/await)
